# Encryption Algorithm RGR — Speck-128 / CBC

Программное обеспечение для шифрования и дешифрования данных с использованием блочного алгоритма **Speck** в режиме **CBC (Cipher Block Chaining)** с поддержкой дополнения блоков по стандарту **PKCS#7**. 

Проект разработан в рамках расчётно-графической работы (РГР) по дисциплине «Программирование» в соответствии с требованиями ГОСТ 19.201-78.

## Особенности реализации
- **Архитектура без классов (POD-ориентированная):** В соответствии с ТЗ, все внутренние состояния, ключевые расписания и контексты библиотек хранятся в виде чистых структур данных (POD). Вся логика реализована через свободные функции.
- **Динамическая модульность:** Алгоритм Speck вынесен в отдельную динамически подключаемую библиотеку (`.dll` / `.so`) со стандартным C-совместимым интерфейсом (`extern "C"`). Клиентская часть загружает библиотеку на лету во время выполнения (Runtime Dynamic Loading).
- **Параметры шифра:** Размер блока зафиксирован на 128 бит (16 байт). Поддерживается генерация и использование ключей переменной длины: 128 бит (16 байт), 192 бита (24 байта) и 256 бит (32 байта).
- **Файловый режим с сохранением метаданных:** При шифровании файлов в начало шифротекста упаковывается заголовок с оригинальным именем и расширением исходного файла, что позволяет корректно восстанавливать файлы при дешифровании.
