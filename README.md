# РГР: Шифрование и дешифрование файлов и текста

Консольное приложение на C++, реализующее шифрование/дешифрование текста и
файлов несколькими симметричными блочными алгоритмами. Каждый алгоритм
собирается в отдельную динамическую библиотеку (`.dll` / `.so`) и
подключается к главной программе в рантайме.

## Архитектура

Каждый шифр разбит на три слоя:

- **`algo/`** — чистая реализация алгоритма (C++ класс). Не знает о консоли,
  файлах или динамических библиотеках. Попадает только в `.dll`/`.so`.
- **`capi/`** — C-совместимая обёртка (`extern "C"`) над классом из `algo/`.
  Необходима потому что классы C++ нельзя безопасно передавать через границу
  динамической библиотеки — ABI разных компиляторов несовместим. Тоже попадает
  только в `.dll`/`.so`.
- **`ui/`** — консольный интерфейс: меню, ввод текста, работа с файлами,
  генератор ключей. Загружает соответствующую `.dll`/`.so` в рантайме через
  `core/module_loader` и вызывает функции из `capi/` через указатели.
  Попадает только в `app.exe`.

`core/module_loader` — единственный файл, общий для всех шифров: кросс-
платформенная обёртка над `LoadLibrary`/`GetProcAddress` (Windows) и
`dlopen`/`dlsym` (Linux/Mac).

## Сборка

Рекомендуемый способ — через CMake, собирает все библиотеки и главную
программу одной командой:

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

После сборки в папке `build/` будут лежать `app.exe`/`app` и все `.dll`/`.so`
рядом друг с другом — готовы к запуску.

Подробные инструкции по сборке отдельных модулей через `g++` — в README
каждого шифра.
